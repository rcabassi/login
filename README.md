🔧 Cambios y clases de Bootstrap usadas
Layout y centrado
d-flex, align-items-center, min-height: 100vh → centra verticalmente

col-md-5 col-lg-4 → mejor ancho en distintas pantallas
Fondo
bg-light → fondo suave
Card
shadow-lg → sombra más elegante
border-0 → elimina borde duro
card-header bg-primary text-white text-center → encabezado destacado
Texto
font-weight-bold → etiquetas más visibles
text-muted, small → pie de tarjeta discreto
Inputs
form-control-lg → campos más grandes y modernos
placeholder → guía visual
Botón
btn-lg → más prominente
btn-block → ocupa todo el ancho
mt-3 → separación superior
Espaciado
p-4 → padding interno cómodo
